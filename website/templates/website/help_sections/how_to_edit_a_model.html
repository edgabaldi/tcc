Passos

  1. Acesse a área administrativa do sistema.
  2. Acesse menu Opções > Modelo
  3. Na listagem de modelos, clique no nome do modelo que você deseja alterar.
  4. Aparecerá o formulário de edição do modelo, modique os dados necessários e clique no botão Salvar, para guardar sua alteração.
