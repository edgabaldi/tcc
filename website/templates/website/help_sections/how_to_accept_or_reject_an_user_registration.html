Motivação:

Todo cadastro realizado no site deve ser avaliado, para evitar problemas legais durante a regularização dos bens arrematados, assim como problemas com a emissão de notas de arrematação. É necessário verificar se todos os dados de documentação e endereço estão corretos. Caso não esteja, o cadastro deve ser rejeitado, até que o usuário confirme as informações solicitadas.

Passos

  1. Acesse a área administrativa do sistema.
  2. Acesse menu opções > Usuários
  3. Na listagem de usuários, selecione um usuário inativo.
  4. No final do formulário de edição, clique no botão Aceitar/Rejeitar Cadastro.
  5. Avalie o cadastro, se tiver dentro dos conformes, clique no botão Ativar caso contrário, informe um motivo da rejeção no campo observações e clique no botão Rejeitar.

Para facilitar a verificação de quais usuários precisam ser avaliados, faça um filtro por usuários inativos.
