Passos

  1. Acesse a área administrativa do sistema.
  2. Acesse menu Opções > Recomendação para Clientes
  3. Aparecerá a listagem de recomendação de produtos recomendados.
  4. No filtro de Usuários, selecione um usuário e clique no botão Filtrar.
  5. Aparecerá os produtos recomendados para o usuário. Quanto mais próximo de 1, o fator de similaridade, mais semelhante ao usuário o produto é.
