Passos

  1. Acesse a área administrativa do sistema.
  2. Acesse menu opções > Usuários
  3. Na listagem de usuários, clique no nome do usuário que você deseja editar.
  4. Aparecerá um formulário com os dados do usuário preenchido.
  5. No final do formulário, possui uma sessão chamada "Grupos", selecione os grupos que você deseja que o usuário faça parte e clique no botão Salvar.
