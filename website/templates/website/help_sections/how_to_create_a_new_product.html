Passos

  1. Acesse a área administrativa do sistema.
  2. Acesse menu Opções > Lotes
  3. Acima do filtro de pesquisa, clique no botão Novo Lote
  4. Aparecerá um formulário com os dados do lote, preencha o formulário e clique no botão Salvar.

Um lote só estará disponível para lance se estiver com o status Liberado para Lance.
